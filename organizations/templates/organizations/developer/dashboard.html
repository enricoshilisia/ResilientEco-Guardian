POST /api/agent/run/ Run the full 5-agent climate risk pipeline JWT required
Runs Monitor → Predict → Decision → Action → Governance agents against live weather data. Returns structured JSON with risk scores, alert level, and agent chain metadata.
Parameters
FieldTypeRequiredDescription
querystringREQNatural language climate query
latfloatREQLatitude (-90 to 90)
lonfloatREQLongitude (-180 to 180)
location_namestringREQHuman-readable location label
org_idstringOPTOrganisation UUID for context-aware analysis
Response
{
  "success": true,
  "result": {
    "monitor": "…", "predict": "…", "decision": "…",
    "predict_data": { "flood_risk": 65, "drought_risk": 12, "heatwave_risk": 20 },
    "decision_data": { "alert_level": "ORANGE", "recommended_actions": ["…"] },
    "agent_chain": [{ "agent": "monitor", "latency_ms": 342 }],
    "session_id": "rg_abc123"
  },
  "latency_ms": 2841
}
